# E-Commerce Retail Sales Analysis

## Project Overview

This project analyzes retail e-commerce transactions using SQL and Power BI. The goal was to identify sales trends, evaluate product performance, assess data quality, and generate actionable business insights.

## Tools Used

* PostgreSQL
* SQL
* Power BI

## Project Workflow

### 1. Data Quality Assessment

* Identified missing Customer IDs
* Checked missing product descriptions
* Investigated negative quantities
* Investigated zero and negative unit prices

### 2. SQL Analysis

* Total Revenue Analysis
* Monthly Revenue Trends
* Top Products Analysis
* Country-Level Analysis

### 3. Dashboard Development

Created an interactive Power BI dashboard featuring:

* Revenue KPIs
* Customer KPIs
* Order KPIs
* Monthly Sales Trends
* Top Countries by Revenue
* Top Products by Revenue

### Key Insights

* Total Revenue: $9.75M
* Total Orders: 541.9K
* Customers: 4K
* Countries Served: 38
* Q4 generated the highest revenue
* November was the best-performing month
